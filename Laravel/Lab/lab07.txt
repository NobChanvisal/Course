Lab 07. Migration

1. Create a new Laravel project Lab_07
2. Create Database name lab_07 and Configure your database in .env at laravel.
3. Run initial migrations to ensure your setup works using : php artisan migrate
4. Create Migration :
	a. creating a categories table to store category id,  names and descriptions.
		- id an unsigned big integer primary key.
		- name as string 
		- description as text that can be null
		- timestamps
	after Run the migration
	b. create a products table for storing product details.
		- id an unsigned big integer primary key.
		- name as string 
		- description as text that can be null
		- price as decimal
		- dis_price as decimal that can be null
		- status as enum : 'available', 'unavailable' default 'available'
		- timestamps
	after Run the migration
	c. add a category_id column to the products table by using 
	   php artisan make:migration add_category_id_to_products_table --table=products
		- Creates an unsigned big integer column and a foreign key constraint referencing 
		  categories.id and Deletes products if their category is deleted.
		- In down(), drop the foreign key first, then the column to avoid errors.
	php artisan migrate
5. Roll back the last migration
6. Roll back all migrations
7. Re-run everything
