1. Create a new controller named TaskController
	- Open app/Http/Controllers/TaskController.php and add an index method
	- In the index method creates an array of tasks and uses compact passing data to view().
	- create index.blade.php inside the resources/views
	- Using @foreach loops to display all task that receive from controller.
	- Open routes/web.php and add this route:
		Route::get('/tasks', [TaskController::class, 'index'])->name('tasks.index');
2. Create a new controller named ProductController
	- Open app/Http/Controllers/ProductController.php and add three method 
		index(): Fetches all products and passes them to a Blade view.
		create(): Returns a simple form view for inputting product details.
		store(): Validates input, adds the product to the array, and redirects back to the index 			with a success message.
	- Define Routes
		GET /products: Calls index() to list products.
		GET /products/create: Calls create() to show the form.
		POST /products: Calls store() to process the form submission.
